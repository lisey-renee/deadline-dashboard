# Deadline Dashboard

A full-stack MERN application built to help students track assignments, prioritize tasks, and manage deadlines with ease.

## Live Demo
**Frontend:** https://deadline-dashboard.vercel.app/
**Backend API:** https://deadline-dashboard.onrender.com

## Key Features
* **Full-Stack Persistence**: Assignments are saved to a MOngoDB cloud database.
* **Optimistic UI**: The interface updates instantly when adding, deleting, or completing tasks.
* **Smart Sorting**: Automatically organizes your workload by due date or priority level.
* **Live Stats**: Real-time counters for pending and finished assignments

## Tech Stack
* **Frontend**: React.js, Tailwind CSS, Axios.
* **Backend**: Node.js, Express.js.
* **Database**: MongoDB Atlas.
* **Deployment**: Vercel (Frontend) and Render (Backend).

## How It Works
1. Add an assignment with a title, course/category, and due date.
2. Select how you wish to sort your tasks, by due date or priority level.
3. Toggle the checkmark when finished or delete tasks when they are no longer needed.
